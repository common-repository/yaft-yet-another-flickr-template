== Description ==

This plugin shows all of your flickr pictures in a page. There's a template you can use to get you wordpress Flickr album. Only fill your info (Flickr api key, secret and so on) and you're in. However, maybe you have to change the template page to your needs.

== Installation ==

1. Upload 'yaft-yet-another-flickr-template' (sorry... I misunderstood plugin naming...) directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy 'showPhotos.php', 'showPhotoPreview.php' and 'ywaft_template.php' in your themes directory
4. Create a new page from the admin panel and assign 'YAFT Template' to it.

== Frequently Asked Questions ==

Q: I get a php error when I try to view the page
A: Check your YAFT options. Maybe you didn't set api key. 

Q: I can't change photos, show next or previous page.
A: That plugin works only if javascript is activated for the browser.
